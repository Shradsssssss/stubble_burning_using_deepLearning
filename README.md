🌾 Stubble Burning Detection Using Deep Learning
📌 Project Overview

Stubble burning is a major environmental issue in agricultural regions, especially in countries like India. Farmers often burn crop residues after harvesting to quickly prepare fields for the next crop cycle. This practice releases large amounts of smoke and harmful pollutants, contributing to severe air pollution and health problems.

This project aims to detect stubble burning areas using satellite imagery and deep learning techniques. By analyzing satellite images, the system can identify regions where crop residue burning has occurred and help in environmental monitoring.

🎯 Objectives

Detect stubble burning areas using satellite imagery

Apply deep learning models for image analysis

Identify and highlight burning regions in agricultural fields

Support environmental monitoring and pollution control

⚙️ Methodology

Collect satellite imagery from remote sensing sources

Preprocess images using geospatial tools

Train deep learning models to detect burning areas

Apply semantic segmentation using U-Net architecture

Analyze and visualize detected stubble burning regions
